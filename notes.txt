to create venv called env:
    python -m venv env

to activate env:
    env/Scripts/activate

once activated:
    pip install flask
    pip install flask-sqlalchemy    - if you want to use it instead of sqlite 3

to deactivate env:
    deactivate

to run flask app:
    python app.py
